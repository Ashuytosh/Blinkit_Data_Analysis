## **Dataset Overview**
This project utilizes the **Blinkit Dataset**, consisting of **8523 rows** and **12 columns**.  

**Column Names:**
- Item Fat Content
- Item Identifier  
- Item Type
- Outlet Establishment Year 
- Outlet Identifier 
- Outlet Location Type 
- Outlet Size 
- Outlet Type 
- Item Visibility  
- Item Weight  
- Total Sales 
- Rating 

**KPI Used:**  
**Total Sales**, **Average Sales**, **No. of Items Sold**, and **Average Ratings**
